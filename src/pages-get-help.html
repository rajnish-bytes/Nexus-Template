User Management queries

How do I add a new user?
Go to the 'User Management' section, click 'Add User', fill in the required details, and submit.
How can I change a user's role?
Navigate to 'User Management', select the user, click 'Edit', and update their role before saving.
Can I deactivate a user instead of deleting them?
Yes, you can disable a user account by setting their status to 'Inactive' in the user settings.
How do I reset a user's password?
Admins can reset passwords by selecting the user in 'User Management' and clicking 'Reset Password'. An email will be sent to the user.
Platform

System related queries

Where can I update the system settings?
Navigate to the 'Settings' section in the admin panel to configure system preferences, security options, and more.
How do I enable or disable maintenance mode?
Go to 'Settings' > 'System Configuration' and toggle the maintenance mode option to enable or disable it.
Can I change the default timezone for the system?
Yes, you can update the system timezone in 'Settings' under 'General Preferences'.
How do I configure email notifications?
Navigate to 'Settings' > 'Email Configuration' to set up SMTP settings and customize notification templates.
